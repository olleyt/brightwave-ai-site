    Bring your own material

    Import a lesson

    Paste text from any source — an Educative.io lesson, an article, your own notes. TideLearn turns it into a study note and flashcards, and folds it into your schedule.

    Getting text out of Educative.io

    Educative doesn't offer an export API, and its terms prohibit automated scraping — so this importer works from pasted text, which keeps things clean: it's your paid content, copied by you, for your own study.

    Fastest workflow: open the lesson, select all (⌘A), copy (⌘C), paste here. The parser strips navigation junk automatically.

    AI generation

    This demo uses a built-in heuristic generator. In production, the same pipeline calls Claude to write the key concept, mnemonic, and question set — the interface is identical.

    Have a doc URL?

    Run npm run import-doc -- <url> locally — it fetches the page, generates a full topic (cards + quiz) with Claude, and commits it into the site's curriculum.
